## üéØ Overview

This Power BI dashboard transforms survey data from data professionals into actionable insights for HR departments, hiring managers, career planners, and data professionals themselves. The interactive single-page dashboard features 9 distinct visualizations covering compensation, skills, demographics, geography, and workplace satisfaction.

**Dashboard File:** `DA_PROJECT.pbix`

## ‚ú® Features

- **Interactive Filtering** - Click any data point to filter all visualizations
- **Real-time Analytics** - Dynamic calculations and aggregations
- **Accessible Design** - Built with AccessibleNeutral theme for optimal readability
- **Multi-dimensional Analysis** - Salary, skills, satisfaction, and demographics in one view
- **Professional Presentation** - Clean, modern interface suitable for stakeholder presentations
- **Cross-visual Drill-through** - Seamless navigation between related data points

## üìä Dashboard Visualizations

### 1. **Survey Participation Metrics**
- **Total Survey Takers** - Card visual displaying unique respondent count
- **Average Age** - Demographic overview of survey participants

### 2. **Compensation Analysis**
- **Average Salary by Job Title** - Horizontal bar chart comparing compensation across roles
- **Gender-Based Salary Distribution** - Donut chart analyzing pay equity

### 3. **Skills & Technology**
- **Favorite Programming Languages** - Column chart showing technology preferences segmented by job title

### 4. **Geographic Distribution**
- **Country of Survey Takers** - Treemap visualization showing respondent locations

### 5. **Satisfaction Metrics**
- **Work/Life Balance Happiness** - Gauge chart measuring work-life satisfaction
- **Salary Satisfaction** - Gauge chart tracking compensation satisfaction

## üîç Key Insights

The dashboard enables stakeholders to answer critical questions:

- What is the average salary for different data professional roles?
- Which programming languages are most popular among data professionals?
- How satisfied are data professionals with their work-life balance?
- Are there compensation disparities between genders?
- Where are data professionals geographically concentrated?
- What is the average age of the data professional workforce?
